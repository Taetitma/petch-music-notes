เว็บโน้ตเพลง รวมเพลงวงเพชร

ไฟล์หลัก: index.html

วิธีอัปเดตเพลง:
1. เพิ่มเพลง/โน้ตใน Google Sheet หน้า รวมเพลงวงเพชร
2. เปิดเว็บ แล้วกดปุ่ม อัปเดตจาก Google Sheet
3. ถ้าอยากให้ PDF/JPG ใหม่โชว์ในเว็บทันที ให้วางลิงก์ Google Drive แบบเห็นตัวลิงก์ในช่องเพลง/คอร์ด/โน้ตด้วย

เว็บนี้ยังมีข้อมูลสำรองฝังไว้ ถ้าวันไหน Google Sheet โหลดไม่ได้ เว็บจะยังเปิดข้อมูลเดิมได้
